Q4. Explain the output:

        
        var x = 21;
        var girl = function () {
        console.log(x);
        var x = 20;
       };
        girl ();

OUTPUT : Undefined
The output is "undefined" instead of "21" is due to variable hoisting the variable of "x" is declared inside the "girl" function wherever we declare a variable without assigning any value to it, JS assign its value as "undefined"

Q7. Write a function that accepts parameter n and returns factorial of n

        
        function fact(n) {
            if (n === 0 || n === 1) {
              return 1;
            } else {
              return n * fact(n - 1);
            }
          }
         console.log(fact(7))

OUTPUT : factorial : 5040
